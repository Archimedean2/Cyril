# Prompt Templates

These are copy-paste prompts for the human relay to the external senior dev.

| File | Template | Use when |
|------|----------|----------|
| `TEMPLATE_A_PRE_IMPLEMENTATION_HANDOFF.md` | Template A — Pre-Implementation | After repo recon, before implementing |
| `TEMPLATE_C_POST_IMPLEMENTATION_TEST_PLAN.md` | Template C — Test Plan Request | After implementation, before writing tests |
| `TEMPLATE_B_BREAK_GLASS_ESCALATION.md` | Template B — Break-Glass | When blocked or deeply uncertain |

Fill in the `[paste ... here]` placeholder in each template with the structured report produced by the agent.
